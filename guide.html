📚 仕入れ判定チェッカー

テスター向けガイド（テスト版）

1. これは何？

中古本せどりの「この本、仕入れていい？」に3秒で答えるスマホ用アプリです。バーコードを読むと、Amazonの相場・売れ行きデータ（Keepa）を自動で取得し、経験者の実仕入れデータ約24,000件から作った基準で OK / 条件付きOK / 見送り を理由付きで判定します。

現在テスト中です。判定がおかしい・使いにくい所を見つけたら、ぜひ教えてください（最後の「フィードバックのお願い」参照）。

2. 必要なもの

  • スマホ（iPhone / Android どちらでも）
  • KeepaのAPIキー（下記参照）

APIキーの入手

  1. keepa.com でアカウント登録し、Data API付きプラン（月額制）を契約
  2. ログイン後、keepa.com/#!api を開くと「Private API access key」が表示される
  3. それをコピーしておく（後で設定画面に貼ります）
💡 テスト期間中、主催者からテスト用キーを渡されている場合は契約不要です。そのキーを使ってください（テスト終了後は無効になります）。

3. インストール（1分）

アプリストアは使いません。URLから直接入れます。

iPhone（Safari）

  1. Safariでアプリの URL を開く
  2. 画面下の 共有ボタン（□に↑）をタップ
  3. 「ホーム画面に追加」を選ぶ → 追加
  4. ホーム画面に緑の本のアイコンが増えたら完了

Android（Chrome）

  1. Chromeでアプリの URL を開く
  2. 右上の ⋮ メニュー → 「ホーム画面に追加」（または「アプリをインストール」）

4. 初期設定（1分）

  1. アプリを起動 → 下の ⚙️ 設定 タブ
  2. 一番上の欄に KeepaのAPIキー を貼り付け
  3. 一番下の 「設定を保存」 をタップ
キーはあなたのスマホの中だけに保存されます。外部に送信されることはありません。

手数料・判定基準はそのままでOK（経験者の実績から作った初期値が入っています）。慣れてきたら自分好みに変えられます。

（任意）過去仕入れリストの照合

自分の仕入れ履歴CSVを読み込んでおくと、判定した本を過去に仕入れたことがある場合、「📋 過去仕入れ実績あり: ◯年◯月に¥◯で仕入れ→¥◯で出品」という情報が判定結果に表示されます。「前に扱ったことがある本か」「当時と相場がどう変わったか」が一目でわかります。

  1. せど楽などから仕入れリストをCSVでエクスポート（SKU, JAN, ASIN, タイトル, …, 仕入原価, …, 出品予定価格, …, 仕入日 の列を含む形式）
  2. そのCSVファイルをスマホに保存（LINEやメールで自分に送る→ファイルに保存、が簡単）
  3. アプリの ⚙️設定 →「過去仕入れリスト照合」→ ファイルを選択 でCSVを指定
  4. 「✅ ◯件を読み込みました」と出たら完了
リストのデータもスマホの中だけに保存され、外部には送信されません。リストを更新したいときは、新しいCSVを同じ手順で読み込み直すだけです（上書きされます）。
⚠ 表示される過去実績は「当時の記録」です。相場は変わっているので、判定そのものは常に現在のKeepaデータで行われます。仕入れ履歴がない人・せど楽を使っていない人は、この設定は飛ばしてOKです。

5. 使い方

  1. 店で気になる本を見つけたら、🔍 判定 タブで 「📷 バーコードをスキャン」（初回はカメラの許可を求められるので「許可」）
  2. 本の裏の 上のバーコード（978で始まる方） にかざす → 自動で判定開始
  3. 値札があれば 仕入値 に入力してから判定すると、粗利まで計算されます。空欄なら「いくら以下なら買っていいか」（損益分岐）を出してくれます

カメラが使いにくい場合は、ASIN（10桁）かJAN（13桁の数字）を手入力でもOKです。

6. 判定の見方

✅ 仕入れOK基準クリア。理由欄の粗利・利益率を確認して仕入れ
⚠️ 条件付きOKチャンスだが注意点あり。理由を必ず読む（例：回転が遅い＝売れるまで時間がかかる／出品ゼロのプレミア候補／基準まであと僅か…）
❌ 見送り基準未達。ただし絶対ダメではなく「経験者の型から外れる」の意味。自信のある根拠があれば自己判断で

データ項目のかんたん解説

中古相場いま出ている中古（状態:良い・送料込）の最安値＝あなたの売価の目安
現在ランキング小さいほど売れている本。〜30万位:よく動く / 〜90万位:普通 / 90万位〜:遅い
推定販売数（90日）直近90日で売れたと推定される回数。0なら要注意
実売の痕跡過去に「実際に売れた」証拠の回数。出品価格の記録だけでは売れた証拠にならない、が本アプリの思想です
損益分岐仕入値この金額以下で仕入れられるなら基準クリア。値札と見比べる
Amazon本体「あり」ならAmazon自身が新品を売っている＝中古の値段の上限になる

判定に迷ったら 「📈 Keepaでグラフを見る」 ボタンで元データを自分の目で確認できます。

7. うまく動かないとき

「APIキーを入力してください」設定タブでキーを貼って「設定を保存」を押したか確認
「Keepaエラー」が出るキーの貼り間違い（前後の空白に注意）か、トークン切れ。少し待つと回復します
カメラが起動しないスマホの設定でブラウザのカメラ許可を確認。手入力でも判定できます
商品が見つからないバーコードの下段（192…）を読んでいないか確認。上段（978…）を読む
表示が古い気がするアプリを完全に閉じて、2回開き直すと最新版になります

8. フィードバックのお願い 🙏

テストで一番知りたいのは「判定と、あなたの感覚がズレた場面」です。

  • 「これOKでしょ」と思ったのに見送りになった本
  • 「これはダメでしょ」と思ったのにOKになった本
  • 意味がわからなかった表示・文言
  • 操作でイラッとしたところ
報告するときは ①判定画面のスクショ ②本のASINかJAN ③あなたならどう判断するか の3点セットだと最高です。判定履歴は 🕐履歴 タブにも残っています。

9. 注意事項

  • 判定は過去データに基づく参考情報です。仕入れの最終判断と結果責任はご自身にあります
  • 相場は常に変動します。特に高額品は判定後にKeepaグラフの目視確認を推奨します
  • テスト版のため、予告なく機能が変わることがあります
  • アプリのURL・この取説の無断転載や再配布はご遠慮ください
