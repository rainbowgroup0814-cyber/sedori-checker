📚 仕入れ判定チェッカー中古本せどり用

Keepa API キー 必須

Keepaにログイン → keepa.com/#!api でキーを確認できます（API付きプランの契約が必要）。キーはこの端末のみに保存され、外部には送信されません。

手数料設定（FBA・本）

判定基準（ランク別）

2年半・2.4万件の実仕入れデータの下位10%ライン（実際に許容してきた床）が初期値です。厳しめにしたい人は数値を上げてください。
例外ルール（自動適用）: ①売価1万円以上＆30万位以内なら利益率10%＋粗利¥1,500でOK ②90万位超は「絶版プレミア枠」として長期在庫の注意付きでOK判定

過去仕入れリスト照合 任意

せど楽などのCSV（SKU,JAN,ASIN,...形式）を読み込むと、判定時に過去の仕入れ実績を表示します。データは端末内のみ。
